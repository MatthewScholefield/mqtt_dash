# MQTT Dash

A simple Flutter dashboard for MQTT devices. Control lights, sensors, switches, etc.

## Features

- Connect to any MQTT broker
- Add buttons, displays, and toggles for your devices
- Real-time updates when messages arrive
- Desktop/Linux support

## Setup

```bash
flutter pub get
flutter run
```

## Usage

1. Open app, add your MQTT broker settings
2. Create widgets for your devices
3. Control/monitor stuff

Platform-specific Flutter setup applies.
